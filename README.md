# 🎮 Raptor Arena - Esports Tournament Management System

Raptor Arena is a console-based Esports Tournament Management System developed in C++ using Object-Oriented Programming (OOP) concepts.

The project is designed to manage university gaming events and esports tournaments by providing features for player registration, team management, tournament organization, match scheduling, and winner tracking.

---

## 🚀 Features

### 👤 User Management
- Player Registration
- Admin Management
- Tournament Organizer Functions

### 🎮 Game Management
- PUBG Mobile
- Free Fire
- Chess
- Tekken 7

### 👥 Team Management
- Create Teams
- Assign Team Captain
- Add Team Members
- Squad-based Registration for PUBG & Free Fire

### 🏆 Tournament Management
- Tournament Registration
- Match Scheduling
- Tournament Bracket Generation
- Winner Announcement

### 📊 Analytics & Reports
- Dynamic Leaderboard
- Tournament Statistics Dashboard
- Revenue Tracking

### 💳 Registration System
- Registration Fee Management
- Unique Registration IDs
- Payment Simulation
- Digital Receipts

### 🏅 Achievement System
- Winner Certificate Generation
- Prize Pool Management

### 📁 File Handling
- Store Player Records
- Store Team Records
- Store Tournament Data
- Persistent Data Storage

## 📚 OOP Concepts Implemented

### Encapsulation
Data members are kept private and accessed through public member functions.

### Inheritance
```text
User
├── Admin
├── Player
└── Organizer
```

### Polymorphism
Virtual functions are used for displaying different game information.

### Abstraction
```text
Game
├── PUBG
├── FreeFire
├── Chess
└── Tekken
```

### Composition
Tournament contains:
- Players
- Teams
- Matches
